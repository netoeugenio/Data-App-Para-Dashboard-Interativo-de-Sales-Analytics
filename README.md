# Data-App-Para-Dashboard-Interativo-de-Sales-Analytics
Este projeto é um aplicativo desenvolvido em Python usando Streamlit, com tema visual totalmente personalizado via CSS. Ele utiliza SQLite como banco de dados local para armazenar e consultar informações de forma leve e eficiente. Ideal para dashboards simples, rápidos e com visual profissional.
